# Nhật Ký Phiên Làm Việc - 2026-06-07

## Mục Tiêu

- Rà soát và tiếp tục triển khai Sprint 1: Foundation & Auth.
- Hoàn thiện luồng đăng ký, xác thực email và đăng nhập.
- Khôi phục frontend sau khi xảy ra Git merge conflict.

## Quyết Định Kỹ Thuật

- Không sử dụng Tailwind; tiếp tục dùng CSS thuần và UI Kit hiện tại.
- Chưa triển khai Docker trong Sprint 1.
- Backend tự phát JWT.
- Access token được giữ trong memory.
- Refresh token sử dụng HttpOnly cookie và được rotate.
- Email được gửi qua Resend thay cho SMTP.
- Khi local chưa cấu hình dịch vụ email, backend có thể log đường link để kiểm thử.
- Không lưu secret thật trong file mẫu hoặc commit lên Git.

## Backend

### Đã Rà Soát Và Thực Hiện

- Rà soát cấu hình chạy local và `run-dev.ps1`.
- Làm rõ cách cấu hình `JWT_SECRET`, Supabase và Resend API key.
- Chuyển hướng email service từ SMTP sang Resend.
- Kiểm tra luồng đăng ký tài khoản và dữ liệu user trong Supabase.
- Xác nhận các user mẫu trong database là seed data phục vụ kiểm thử.
- Kiểm tra lỗi kết nối từ frontend tới backend trong luồng đăng ký.

### Lưu Ý Khi Chạy Local

- Backend cần có đầy đủ biến môi trường bắt buộc trước khi khởi động.
- Không đưa API key hoặc database password thật vào Git.
- Nếu thay đổi secret đã từng chia sẻ hoặc commit, cần rotate secret đó.

## Frontend

### Đã Khôi Phục Sau Merge

- Giải quyết conflict tại `AppProviders.jsx`.
- Khôi phục React Router và cơ chế phục hồi phiên khi ứng dụng khởi động.
- Khôi phục các route:
  - `/login`
  - `/register`
  - `/forgot-password`
  - `/reset-password`
  - `/verify-email`
  - `/app`
  - `/account/profile`
  - `/account/change-password`
- Giữ auth implementation dùng access token trong memory và refresh bằng HttpOnly cookie.
- Loại bỏ các service, route và layout trùng không tương thích được kéo vào từ merge.
- Khôi phục `.env.example` với:
  - `VITE_API_BASE_URL`
  - `VITE_GOOGLE_CLIENT_ID`
- Cập nhật `.gitignore` để bỏ qua các file môi trường thật nhưng vẫn theo dõi `.env.example`.

### Chỉnh Sửa Giao Diện

- Giữ nút `Log in` trong suốt, không viền và không có bóng khi hover.
